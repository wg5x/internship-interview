## 技术二面结构

时长：90 分钟。

题目：设计一个“浏览器自动运营 Agent + 通用 Agent 工作台”。

要求候选人现场说明：

- 浏览器自动化：如何登录、保持账号态、处理 Cookie/验证码/动态渲染/反爬、执行抓取/表单/发布任务。
- 任务编排：如何把“选题 -> 内容生成 -> 审核 -> 发布 -> 复盘”拆成可恢复的工作流。
- Agent Runtime：工具注册、schema、上下文构造、记忆、checkpoint/resume、任务取消和并发控制。
- 平台能力：如何支持类似 Claw/WorkBuddy/龙虾的工具扩展、审计日志、权限审批和可观测 trace。
- RAG：如何接入知识库、运营素材库、历史案例库，做检索、重排、引用和评测。
- Coze/工作流：如果用 Coze/Dify 快速搭建，哪些节点适合低代码，哪些必须自研。
- 风险控制：账号风控、敏感操作、重复发布、越权工具、人工确认和回滚。
